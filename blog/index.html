Diversity is desperately needed in IT

May 30, 2021 Â· One min read

Dan Arroyo

Lead Developer

I want to start using this blog to get things off my chest related to how I see the state of IT today. Although I think it has greatly improved since when I started in IT, I believe that diversity in IT is needed. I get looks of surprise when I mention it to people, but once I begin to explain it, many people seem to quickly get it. Development in IT is dominated by White, Indian and Asian males. We really do not understand the experience that women, African Americans (male or female) and other races, genders, etc... go thru. Those people groups need their own representation in the industry.

I guess it is early to blog about this in this blog since I have not set up discussion yet. I would love to hear other people's opinions on this. But, I guess I am taking steps in good directions. Today I am blogging this from a shiny new Ubuntu Machine I just installed and set up for development.

I have set up this blog in the SadSumo GitHub pages. It was pretty straight forward. There were only a couple of gotchas.

  • Put a file called CNAME with the custom domain you want to use in the static folder of the project. "Anything within the static directory will be copied to the root of the build directory for deployment." https://docusaurus.io/docs/deployment
  • I ended up putting all of the code in the master branch and main is blank. Then in the repository's settings in Pages I set the page to be built from the master branch.
  • The master branch only contains the build after running:
command to deploy
cmd /C 'set "GIT_USER=MY_GIT_USERNAME" && yarn deploy'
